Superguest

Implementar una nueva funcionalidad

Para ello creamos una experiencia de usuario para una nueva característica o funcionalidad que queríamos añadir a un producto o servicio existente.

Y nos centramos en la forma en que los usuarios interactúan con la nueva funcionalidad y cómo se integra en el diseño general del producto o servicio.

Investigación y definición

Primero identificamos el problema que estábamos tratando de resolver con la nueva funcionalidad. Es esencial comprender las necesidades y expectativas de los usuarios, así como cualquier limitación técnica o de presupuesto.

El siguiente paso fue realizar una investigación de usuario para comprender cómo los usuarios interactúan con el producto o servicio actual y cómo podrían utilizar la nueva funcionalidad. Esta investigación incluyó entrevistas con usuarios, encuestas, pruebas de usabilidad y análisis de datos de uso existentes.

Prototipado

Una vez que seleccionado un concepto, creamos diseños más detallados en alta fidelidad, que se pareciesen al producto o servicio actual, y que mostrasen cómo se integrará la nueva funcionalidad.

Con la investigación de usuario en mente, creamos varios conceptos de diseño para la nueva funcionalidad. Dibujos a mano, wireframes y prototipos interactivos. Con el objetivo es explorar varias opciones y evaluar cuál es la mejor solución para el problema que estábamos tratando de resolver.
